Este sistema web  implementa un modelo de Inteligencia Artificial que preselecciona candidatos para cubrir un puesto a través de cuatro características, Antigüedad, Idioma, Voluntad Entusiasta, Firmeza y Seguridad. Como resultado arroja si el candidato es Muy Bueno, Bueno o Regular para cubrir la vacante. Se realiza un sistema de predicción y gestión de candidatos utilizando Django, scikit-learn, TensorFlow/Keras, y ReportLab.


Resumen de las implementaciones principales:

 

Predecir Candidato: Una vista que muestra un formulario para ingresar datos de un nuevo candidato. Cuando se envía el formulario, el modelo pre-entrenado se utiliza para predecir la categoría del candidato y se guarda en la base de datos. Luego se muestra el resultado de la predicción junto con la información del candidato y una lista de todos los candidatos almacenados.

Cargar Modelo y Definición de Vistas: El modelo de red neuronal pre-entrenado se carga desde un archivo `.h5`. Luego, se definen varias vistas utilizando Django, cada una con una funcionalidad específica.

Estadísticas de Candidatos: Una vista que genera estadísticas sobre los candidatos almacenados en la base de datos, incluyendo un gráfico de barras que muestra el número de candidatos en diferentes categorías (Regular, Bueno, Muy Bueno). Las estadísticas y el gráfico se muestran en una plantilla HTML.

Generar PDF: Una vista que genera un documento PDF con información detallada sobre todos los candidatos almacenados en la base de datos, incluyendo sus datos personales y la distribución de categorías. También incluye un gráfico de torta que muestra la distribución de categorías.

Borrar Candidatos: Una función para borrar todos los candidatos almacenados en la base de datos cuando se realiza una solicitud POST.

En resumen, este código proporciona funcionalidades completas para la gestión de candidatos, incluyendo la predicción de categorías y la generación de informes en PDF utilizando Django y bibliotecas de Python.
